A list of useful resources to learn web development:

  • SoloLearn : a good starting point for beginners.
  • FreeCodeCamp : another nice place for newcomers.
  • Codecademy : as good as the two first ones.
  • w3schools : an essential and really exhaustive resource!
  • TheOdinProject : learn the skills you need to get hired as a web developer.
  • CodeWars : learn by practising.
